# 🚀 Menu Lateral (CSS-Only Sidebar)

> Um componente de navegação lateral expansível construído com uma abordagem "CSS-only", desenvolvido como parte do meu portfólio de estudos em Programação Full-Stack.

## 📖 Sobre o Projeto

Este projeto consiste em um **Menu Lateral (Sidebar)** moderno. O grande diferencial técnico deste componente é a ausência de JavaScript para o controle de estado (abrir/fechar). Em vez disso, foi utilizada a técnica do **Checkbox Hack** (`<input type="checkbox">` combinado com a tag `<label>`), demonstrando domínio avançado sobre seletores CSS estruturais e comportamento de formulários no HTML5.

O menu inclui ícones de navegação intuitivos e uma seção dedicada a links de redes sociais, proporcionando uma interface limpa e amigável (User-Friendly).

## ✨ Funcionalidades e Aprendizados

* **Toggle sem JavaScript:** Uso inteligente do `checkbox` invisível para gerenciar o estado do menu (Aberto/Fechado).
* **Integração de Ícones:** Utilização da biblioteca **Font Awesome (v5.15.2)** para enriquecer a interface visual.
* **Semântica HTML5:** Estruturação correta utilizando tags semânticas como `<nav>`, `<ul>` e `<li>`.
* **Design Limpo:** Estrutura pronta para receber estilizações CSS personalizadas (`css/style.css`), com base em transições suaves e layout flexível.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Estilização, transições e lógica de estado (Checkbox Hack).
* **Font Awesome:** Tipografia de ícones vetorizados.
